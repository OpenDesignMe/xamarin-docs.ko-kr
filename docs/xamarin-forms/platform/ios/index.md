# <a name="ios-platform-features"></a>iOS 플랫폼 기능

## <a name="ios-specific-formatting"></a>iOS 별 서식 지정

Xamarin.Forms는 플랫폼 간 사용자 인터페이스 스타일 및 색을 설정할 수 있습니다도 iOS 프로젝트에 플랫폼별 Api를 사용 하 여 iOS의 테마를 설정 하기 위한 다른 옵션이 있습니다.

[자세히 알아보세요](theme.md) 와 같은 특정 iOS Api를 사용 하 여 사용자 인터페이스 형식에 대 한 **Info.plist** 구성 및 `UIAppearance` API.
