# <a name="xamarinforms-templates"></a>Xamarin.Forms 템플릿

## <a name="control-templatescontrol-templatesindexmd"></a>[컨트롤 템플릿](control-templates/index.md)

Xamarin.Forms 컨트롤 템플릿은 런타임에 응용 프로그램 페이지 쉽게 테마와 다시 테마 수를 제공합니다.
